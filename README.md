## Quick Start

### Prerequisites

* Node.js
* Git

### Clone and Run

```bash
git clone https://github.com/harinisrikb/portfolio.git
cd portfolio
npm install
npm run dev
```

Your app will be available at:

```text
http://localhost:5173
```
